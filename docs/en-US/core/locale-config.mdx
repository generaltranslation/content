## Overview

`LocaleConfig` is a lightweight class that provides locale formatting utilities and locale resolution helpers. It's designed as a standalone import for projects that only need formatting and locale methods — without pulling in the full [`GT`](/docs/core) class and its translation dependencies.

This gives you better tree-shaking and a smaller bundle when you don't need translation functionality.

```typescript
import { LocaleConfig } from 'generaltranslation';

const localeConfig = new LocaleConfig({
  defaultLocale: 'en',
  locales: ['es', 'fr', 'ja'],
});
```

If you already have a `GT` instance, you can access its `LocaleConfig` via `gt.localeConfig`.

---

## Constructor

```typescript
new LocaleConfig(params?: LocaleConfigConstructorParams)
```

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `defaultLocale` | `string` | `'en'` | Fallback locale for formatting |
| `locales` | `string[]` | `[]` | Supported locales |
| `customMapping` | [`CustomMapping`](/docs/core/types/custom-mapping) | — | Custom locale alias mapping |

## Formatting methods

| Method | Description |
|--------|-------------|
| [`formatNum`](/docs/core/format/numbers) | Format numbers via `Intl.NumberFormat` |
| [`formatDateTime`](/docs/core/format/dates) | Format dates via `Intl.DateTimeFormat` |
| [`formatCurrency`](/docs/core/format/currency) | Format currency values |
| [`formatRelativeTime`](/docs/core/format/relative-time) | Format relative time (e.g. "3 days ago") |
| [`formatRelativeTimeFromDate`](/docs/core/format/relative-time) | Compute and format relative time from a `Date` |
| [`formatList`](/docs/core/format/lists) | Format arrays as human-readable lists |
| [`formatListToParts`](/docs/core/format/lists) | Like `formatList`, but returns parts |
| [`formatMessage`](/docs/core/format/messages) | Format ICU message strings with variables |
| `formatCutoff` | Truncate a string with cutoff options |

---

## Locale methods

| Method | Description |
|--------|-------------|
| `getLocaleName(locale)` | Display name of a locale (e.g. `"Japanese"`) |
| `getLocaleEmoji(locale)` | Flag emoji for a locale (e.g. `"🇫🇷"`) |
| [`getLocaleProperties(locale)`](/docs/core/types/locale-properties) | Locale metadata object |
| `getLocaleDirection(locale)` | `'ltr'` or `'rtl'` |
| `isValidLocale(locale)` | Check if a locale string is valid |
| `standardizeLocale(locale)` | Canonicalize to BCP 47 (e.g. `en_US` → `en-US`) |
| `resolveCanonicalLocale(locale)` | Resolve aliases to canonical BCP 47 |
| `resolveAliasLocale(locale)` | Resolve canonical locale back to alias |
| `isSameDialect(...locales)` | Check if locales are the same dialect |
| `isSameLanguage(...locales)` | Check if locales share a language |
| `isSupersetLocale(super, sub)` | Check if one locale is a superset of another |
