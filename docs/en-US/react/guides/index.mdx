Browse how-to Guides for General Translation across React, Next.js, TanStack Start, and React Native. Each guide covers all four frameworks and shows per-framework differences in tabs.

- [Configure General Translation](/docs/react/guides/configure): Set up initialization, the provider, credentials, and translation delivery for your framework.
- [Translate JSX](/docs/react/guides/translate-jsx): Translate JSX content in place with the General Translation `<T>` component.
- [Translate strings](/docs/react/guides/translate-strings): Translate standalone strings such as labels and placeholders with the `useGT` hook and Next.js `getGT`.
- [Format numbers and dates](/docs/react/guides/format-variables): Insert locale-aware numbers, currencies, dates, and variables into translated content.
- [Handle plurals and branches](/docs/react/guides/plurals-and-branches): Render count-based and conditional content with the `<Plural>` and `<Branch>` components.
- [Manage locales](/docs/react/guides/manage-locales): Let users switch languages and read the active locale.
- [Translate with dictionaries](/docs/react/guides/dictionaries): Translate strings from a central dictionary with the `useTranslations` hook.
- [Store translations locally](/docs/react/guides/local-translations): Bundle translations in your app instead of loading them from the General Translation CDN.
- [Lint your code](/docs/react/guides/lint-your-code): Catch common internationalization mistakes with the General Translation ESLint plugin.
