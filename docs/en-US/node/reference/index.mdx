Browse Reference pages for the General Translation `gt-node` library. `gt-node` is a thin server layer that adds Node.js setup and request-locale helpers on top of the `gt-i18n` translation functions it re-exports.

## Configuration [#configuration]

| Page | Description |
| --- | --- |
| [Configuration](/docs/node/reference/config) | Keys accepted by `initializeGT`. |

## Setup [#setup]

| Page | Description |
| --- | --- |
| [initializeGT](/docs/node/reference/functions/initialize-gt) | Configure `gt-node` once at server startup. |
| [withGT](/docs/node/reference/functions/with-gt) | Bind a locale for the duration of a request. |
| [getRequestLocale](/docs/node/reference/functions/get-request-locale) | Detect the best supported locale from a request. |

## Translation functions [#translation-functions]

| Page | Description |
| --- | --- |
| [getGT](/docs/node/reference/functions/get-gt) | Get a translation function for the request locale. |
| [getMessages](/docs/node/reference/functions/get-messages) | Get a resolver for messages registered with `msg`. |
| [getTranslations](/docs/node/reference/functions/get-translations) | Get a dictionary resolver for the request locale. |
| [tx](/docs/node/reference/functions/tx) | Translate a string on demand at runtime. |
| [msg](/docs/node/reference/functions/msg) | Register a string for translation at module scope. |
