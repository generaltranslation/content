---

title: Integrations
description: Use General Translation with third-party content platforms and plugins.

---

Connect General Translation to third-party content platforms so you can translate content where it already lives. Each integration is a plugin with its own Quickstart, Guides, and Reference.

- [Google Drive](/docs/integrations/google-drive/quickstart): Translate Google Docs and Google Slides from the General Translation Dashboard with personal or Organization shared-drive access.
- [Mintlify](/docs/integrations/mintlify/quickstart): Translate a Mintlify documentation site through GitHub with Locadex, the General Translation AI agent — no plugin to install.
- [Sanity](/docs/integrations/sanity/quickstart): Translate Sanity documents from inside Sanity Studio with the `gt-sanity` plugin.
- [Storyblok](/docs/integrations/storyblok/quickstart): Translate Storyblok stories from the General Translation Dashboard by connecting a space with credentials.
