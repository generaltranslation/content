Set up and run Sanity translation workflows with General Translation.

- [Configure Sanity](/docs/integrations/sanity/guides/configure-sanity): Configure the General Translation gt-sanity plugin locales, document filters, credentials, language fields, and singletons.
- [Translate content](/docs/integrations/sanity/guides/translate-content): Translate Sanity documents with General Translation from the document action, the tab view, or the Translations page.
- [Manage translations](/docs/integrations/sanity/guides/manage-translations): Manage General Translation Sanity translations with imports, reference patching, bulk publishing, and field behavior.
- [Query translations](/docs/integrations/sanity/guides/query-translations): Query translated Sanity documents with GROQ using the General Translation document-level localization model.
