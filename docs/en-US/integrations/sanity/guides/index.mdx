Set up and run Sanity translation workflows with General Translation.

- [Configuring Sanity](/docs/integrations/sanity/guides/configuring-sanity): Configure the General Translation gt-sanity plugin locales, document filters, credentials, language fields, and singletons.
- [Translating content](/docs/integrations/sanity/guides/translating-content): Translate Sanity documents with General Translation from the document action, the tab view, or the Translations page.
- [Managing translations](/docs/integrations/sanity/guides/managing-translations): Manage General Translation Sanity translations with imports, reference patching, bulk publishing, and field behavior.
- [Querying translations](/docs/integrations/sanity/guides/querying-translations): Query translated Sanity documents with GROQ using the General Translation document-level localization model.
