Browse how-to Guides for the General Translation CLI.

- [Configure the CLI](/docs/cli/guides/configure): Set up a General Translation gt.config.json with your locales, files, and storage options.
- [Generate translations](/docs/cli/guides/generate-translations): Translate your project with the General Translation CLI and save the results to your codebase or the CDN.
- [Manage translations](/docs/cli/guides/manage-translations): Split the General Translation workflow into upload, enqueue, and download stages, stage translations for review, and sync local edits.
- [Track translations by branch](/docs/cli/guides/branching): Track translations separately per git branch with General Translation while you work on features.
- [Use auto-derived translations](/docs/cli/guides/autoderive): Automatically generate a separate translation entry for each interpolated value with Autoderive in the General Translation CLI.
- [Use automatic JSX injection](/docs/cli/guides/auto-jsx): Wrap translatable JSX text in General Translation components at build time with automatic JSX injection.
