Browse how-to Guides for the General Translation CLI.

- [Configuring the CLI](/docs/cli/guides/configuring): Set up a General Translation gt.config.json with your locales, files, and storage options.
- [Generating translations](/docs/cli/guides/generating-translations): Translate your project with the General Translation CLI and save the results to your codebase or the CDN.
- [Managing translations](/docs/cli/guides/managing-translations): Split the General Translation workflow into upload, enqueue, and download stages, stage translations for review, and sync local edits.
- [Tracking translations by branch](/docs/cli/guides/branching): Track translations separately per git branch with General Translation while you work on features.
- [Using auto-derived translations](/docs/cli/guides/using-autoderive): Automatically generate a separate translation entry for each interpolated value with Autoderive in the General Translation CLI.
- [Using automatic JSX injection](/docs/cli/guides/using-auto-jsx): Wrap translatable JSX text in General Translation components at build time with automatic JSX injection.
