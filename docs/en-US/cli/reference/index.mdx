## Global flags [#global-flags]

These flags are registered on the root `gt` program and apply to **every** command.

| Flag | Description | Type | Default |
| --- | --- | --- | --- |
| `--skip-version-check` | Skip the monorepo consistency check that verifies your General Translation packages share the same version. | `boolean` | `false` |

*Note: you can skip the version check permanently by setting `"skipVersionCheck": true` in `gt.config.json` instead of passing the flag on every run.*
