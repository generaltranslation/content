Browse Reference pages for the General Translation Python SDK.

*Note: The Python SDK is experimental — `gt-i18n`, `gt-flask`, and `gt-fastapi` are at version 0.3.0 and require Python 3.10 or later.*

| Page | Description |

## Functions [#functions]

| Page | Description |
| --- | --- |
| [initialize_gt](/docs/python/reference/functions/initialize-gt) | Configure the SDK once for a Flask or FastAPI app. |
| [t](/docs/python/reference/functions/t) | Translate and interpolate an ICU MessageFormat string. |
| [t_fallback](/docs/python/reference/functions/t-fallback) | Interpolate a string without a translation lookup. |
| [msg](/docs/python/reference/functions/msg) | Register a message with its options as an encoded string. |
| [m_fallback](/docs/python/reference/functions/m-fallback) | Resolve an encoded message, or interpolate a plain string. |
| [interpolate_message](/docs/python/reference/functions/interpolate-message) | Interpolate variables into an ICU MessageFormat string. |
| [hash_message](/docs/python/reference/functions/hash-message) | Hash an ICU MessageFormat string for translation lookup. |
| [extract_variables](/docs/python/reference/functions/extract-variables) | Filter reserved GT keys to get user variables. |
| [decode_msg](/docs/python/reference/functions/decode-msg) | Extract the message from an encoded [`msg`](/docs/python/reference/functions/msg) result. |
| [decode_options](/docs/python/reference/functions/decode-options) | Extract the options dict from an encoded `msg` result. |
| [declare_var](/docs/python/reference/functions/declare-var) | Mark a dynamic value as a non-translatable variable. |
| [derive](/docs/python/reference/functions/derive) | Mark content with a finite set of variants for extraction. |
| [declare_static](/docs/python/reference/functions/declare-static) | Deprecated alias for [`derive`](/docs/python/reference/functions/derive). |
| [decode_vars](/docs/python/reference/functions/decode-vars) | Replace variable markers in a string with their values. |
| [get_locale](/docs/python/reference/functions/get-locale) | Read the active locale for the current request. |
| [get_locales](/docs/python/reference/functions/get-locales) | List the supported locales. |
| [get_default_locale](/docs/python/reference/functions/get-default-locale) | Read the configured source locale. |
| [get_version_id](/docs/python/reference/functions/get-version-id) | Read the pinned translation version id. |
| [get_i18n_manager](/docs/python/reference/functions/get-i18n-manager) | Access the active [`I18nManager`](/docs/python/reference/classes/i18n-manager). |
| [set_i18n_manager](/docs/python/reference/functions/set-i18n-manager) | Register the active `I18nManager`. |
