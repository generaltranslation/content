Browse how-to Guides for the General Translation Python SDK.

- [Translating strings](/docs/python/guides/translating-strings): Translate strings with variables, context, and options using the General Translation [`t`](/docs/python/reference/functions/t) and [`msg`](/docs/python/reference/functions/msg) functions.
- [Detecting the request locale](/docs/python/guides/detecting-locale): Resolve each request's locale automatically or with your own logic in General Translation.
- [Storing translations locally](/docs/python/guides/storing-translations): Load translations from local files instead of the General Translation CDN.
- [Declaring variables and variants](/docs/python/guides/declaring-variables): Preserve dynamic values and generate finite variants with General Translation [`declare_var`](/docs/python/reference/functions/declare-var) and [`derive`](/docs/python/reference/functions/derive).
