## Overview

The `declare_var` function marks a value as a non-translatable variable within an ICU MessageFormat string.
This ensures that the value is preserved as-is during translation.

```python
from gt_fastapi import declare_var, t  # or from gt_flask / generaltranslation

message = t(
    "Welcome to {brand}!",
    brand=declare_var("General Translation", name="brand"),
)
```

## Reference

### Parameters

| Name | Type | Description |
|---|---|---|
| `variable` | `str \| int \| float \| bool \| None` | The value to embed. |
| `name` | `str \| None` | Optional human-readable variable name for translation context. |

### Returns

`str` — An ICU `{_gt_, select, ...}` construct that wraps the value.

---

## Example

### Brand names

```python
from gt_fastapi import declare_var, t

message = t(
    "Powered by {company}",
    company=declare_var("General Translation", name="company"),
)
```

The translator sees that `company` is a variable and will not attempt to translate "General Translation".

### Numeric values

```python
from gt_fastapi import declare_var, t

message = t(
    "Order #{order_id} confirmed",
    order_id=declare_var(12345, name="order_id"),
)
```
