## Overview

The `generaltranslation` package provides standalone formatting functions that can be used without the [`GT`](/docs/python/api/gt) class.
These are the same functions available as methods on the `GT` class, but they accept explicit locale parameters.

```python
from generaltranslation import format_num, format_currency, format_date_time, format_list
```

---

## `format_num`

Format a number according to a locale.

```python
from generaltranslation import format_num

format_num(1234567, locales="en")         # "1,234,567"
format_num(1234567, locales="de")         # "1.234.567"
format_num(0.75, locales="en", options={"style": "percent"})  # "75%"
format_num(1234, locales="en", options={"notation": "compact"})  # "1K"
```

### Parameters

| Name | Type | Default | Description |
|---|---|---|---|
| `value` | `int \| float` | — | The number to format. |
| `locales` | `str \| list[str] \| None` | `"en"` | BCP 47 locale tag(s). |
| `options` | `dict \| None` | `None` | Formatting options. |

### Options

| Name | Type | Description |
|---|---|---|
| `style` | `str` | `"decimal"` (default) or `"percent"`. |
| `notation` | `str` | `"standard"` (default) or `"compact"`. |
| `use_grouping` | `bool` | Whether to use grouping separators. Default `True`. |
| `minimum_fraction_digits` | `int` | Minimum number of fraction digits. |
| `maximum_fraction_digits` | `int` | Maximum number of fraction digits. |
| `minimum_integer_digits` | `int` | Minimum number of integer digits. |

## `format_currency`

Format a currency value according to a locale.

```python
from generaltranslation import format_currency

format_currency(29.99, "USD", locales="en")  # "$29.99"
format_currency(29.99, "EUR", locales="de")  # "29,99\xa0€"
```

### Parameters

| Name | Type | Default | Description |
|---|---|---|---|
| `value` | `int \| float` | — | The currency amount. |
| `currency` | `str` | `"USD"` | ISO 4217 currency code. |
| `locales` | `str \| list[str] \| None` | `"en"` | BCP 47 locale tag(s). |
| `options` | `dict \| None` | `None` | Formatting options. |

### Options

| Name | Type | Description |
|---|---|---|
| `currency_display` | `str` | `"symbol"` (default), `"name"`, `"code"`, or `"narrowSymbol"`. |
| `currency_sign` | `str` | `"standard"` (default) or `"accounting"`. |

## `format_date_time`

Format a date/time value according to a locale.

```python
from datetime import datetime
from generaltranslation import format_date_time

format_date_time(datetime.now(), locales="en", options={"date_style": "long"})
# "March 7, 2026"

format_date_time(datetime.now(), locales="ja", options={"date_style": "full"})
# "2026年3月7日土曜日"
```

### Parameters

| Name | Type | Default | Description |
|---|---|---|---|
| `value` | `datetime \| str` | — | A `datetime` object or ISO 8601 string. |
| `locales` | `str \| list[str] \| None` | `"en"` | BCP 47 locale tag(s). |
| `options` | `dict \| None` | `None` | Formatting options. |

### Options

| Name | Type | Description |
|---|---|---|
| `date_style` | `str` | `"full"`, `"long"`, `"medium"`, or `"short"`. |
| `time_style` | `str` | `"full"`, `"long"`, `"medium"`, or `"short"`. |

If both are provided, both date and time are formatted. If only one is provided, only that part is formatted. If neither, `"medium"` datetime is used.

## `format_list`

Format a list of items with locale-aware conjunctions.

```python
from generaltranslation import format_list

format_list(["Alice", "Bob", "Charlie"], locales="en")
# "Alice, Bob, and Charlie"

format_list(["Alice", "Bob", "Charlie"], locales="en", options={"type": "disjunction"})
# "Alice, Bob, or Charlie"
```

### Parameters

| Name | Type | Default | Description |
|---|---|---|---|
| `value` | `list` | — | The list of items to format. |
| `locales` | `str \| list[str] \| None` | `"en"` | BCP 47 locale tag(s). |
| `options` | `dict \| None` | `None` | Formatting options. |

### Options

| Name | Type | Description |
|---|---|---|
| `type` | `str` | `"conjunction"` (default), `"disjunction"`, or `"unit"`. |
| `style` | `str` | `"long"` (default), `"short"`, or `"narrow"`. |

## `format_relative_time`

Format a relative time value.

```python
from generaltranslation import format_relative_time

format_relative_time(-3, "day", locales="en")   # "3 days ago"
format_relative_time(1, "hour", locales="en")    # "in 1 hour"
format_relative_time(-2, "month", locales="es")  # "hace 2 meses"
```

### Parameters

| Name | Type | Default | Description |
|---|---|---|---|
| `value` | `int \| float` | — | The relative time value. Negative for past, positive for future. |
| `unit` | `str` | — | Time unit: `"second"`, `"minute"`, `"hour"`, `"day"`, `"week"`, `"month"`, `"year"`, `"quarter"`. |
| `locales` | `str \| list[str] \| None` | `"en"` | BCP 47 locale tag(s). |

## `format_message`

Format an ICU MessageFormat string with variables.

```python
from generaltranslation import format_message

format_message(
    "{count, plural, =0 {No items} =1 {One item} other {{count} items}}",
    locales="en",
    variables={"count": 5},
)
# "5 items"
```

### Parameters

| Name | Type | Default | Description |
|---|---|---|---|
| `message` | `str` | — | The ICU message pattern. |
| `locales` | `str \| list[str] \| None` | `"en"` | BCP 47 locale tag(s) for plural rule selection. |
| `variables` | `dict \| None` | `None` | Variable values to substitute. |
