## Overview

The `t` function is the primary translation function. It looks up a translation for the current locale, interpolates variables, and returns the result.

```python
from gt_fastapi import t  # or from gt_flask import t

translated = t("Hello, world!")
```

If no translation is found for the current locale, the source string is returned.

<Callout>
  **Setup Required:**
  [`initialize_gt`](/docs/python/api/initialize-gt) must be called before using `t`.
</Callout>

## Reference

### Parameters

| Name | Type | Description |
|---|---|---|
| `message` | `str` | The ICU MessageFormat source string to translate. |
| `**kwargs` | `object` | Interpolation variables and GT options. |

### GT options (special kwargs)

| Name | Type | Description |
|---|---|---|
| `$context` | `str` | Additional context to disambiguate translations with the same source text. |
| `$id` | `str` | Custom identifier for the translation entry. |
| `$max_chars` | `int` | Maximum character count hint for the translation. |

All other keyword arguments are treated as interpolation variables.

### Returns

`str` — The translated and interpolated string.

---

## Behavior

1. The current locale is read from the `I18nManager` (set by the middleware from [`initialize_gt`](/docs/python/api/initialize-gt)).
2. If the current locale matches the default locale, the source string is returned with variables interpolated.
3. Otherwise, a translation is looked up by hash. If found, the translated string is interpolated and returned.
4. If no translation is found, the source string is returned with variables interpolated.

---

## Examples

### Basic usage

```python
from gt_fastapi import t

greeting = t("Hello, world!")
```

### With variables

```python
from gt_fastapi import t

greeting = t("Hello, {name}!", name="Alice")
```

The variable `name` is interpolated but not translated.

### ICU MessageFormat (plurals)

```python
from gt_fastapi import t

message = t(
    "{count, plural, =0 {No items} =1 {One item} other {{count} items}}",
    count=5,
)
# "5 items" (or translated equivalent)
```

### With context

```python
from gt_fastapi import t

# "Bank" as in a river bank vs. a financial bank
label = t("Bank", **{"$context": "financial institution"})
```
