## Overview

The `t_fallback` function interpolates variables into a message string without performing any translation lookup.
It is useful for rendering the source string with variables when you explicitly do not want translation.

```python
from gt_i18n import t_fallback

result = t_fallback("Hello, {name}!", name="Alice")
# "Hello, Alice!"
```

## Reference

### Parameters

| Name | Type | Description |
|---|---|---|
| `message` | `str` | The ICU MessageFormat string to interpolate. |
| `**kwargs` | `object` | Interpolation variables. |

### Returns

`str` — The interpolated string (not translated).

---

## Example

```python
from gt_i18n import t_fallback

# Always returns the source string with variables filled in
message = t_fallback(
    "{count, plural, =0 {No items} =1 {One item} other {{count} items}}",
    count=3,
)
# "3 items"
```
