## Overview

`declare_static` wraps a static value with a GT variable marker. Unlike [`declare_var`](/docs/python/api/declare-var), it is intended for values known at build time.

```python
from gt_flask import declare_static  # or from gt_fastapi import declare_static

brand = declare_static("brand", "General Translation")
```

## Notes

- Use `declare_static` for values that don't change (e.g., brand names, product names).
- See also [`declare_var`](/docs/python/api/declare-var) for runtime values.
