## Overview

`decode_vars` extracts GT variable markers from a string and returns the decoded variable map.

```python
from gt_flask import decode_vars  # or from gt_fastapi import decode_vars

variables = decode_vars(marked_string)
```

## Reference

### Parameters

<TypeTable
  type={{
    "text": {
      type: 'str',

    },
  }}
/>

### Returns

`dict[str, str]` — a mapping of variable names to their values.
