## Overview

The `decode_vars` function replaces GT variable constructs (`{_gt_, select, other {value}}`) in an ICU MessageFormat string with their embedded values.

```python
from generaltranslation import decode_vars

original = decode_vars(icu_string_with_gt_vars)
```

This is the inverse of [`declare_var`](/docs/python/api/declare-var) — it extracts the raw values from the GT variable wrappers.

## Reference

### Parameters

| Name | Type | Description |
|---|---|---|
| `icu_string` | `str` | An ICU MessageFormat string potentially containing GT variable constructs. |

### Returns

`str` — The string with GT variable selects replaced by their embedded values.

---

## Example

```python
from generaltranslation import declare_var, decode_vars

# Create a variable-wrapped string
var = declare_var("General Translation", name="brand")
# var = "{_gt_, select, other {General Translation} _gt_n_ {brand}}"

# Decode it back
decoded = decode_vars(var)
# decoded = "General Translation"
```
