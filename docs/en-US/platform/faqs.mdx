### What is the General Translation platform?

The [General Translation platform](https://www.generaltranslation.com) is a dashboard for managing your translations, API keys, and AI context settings like glossaries and style controls.

### How do I get an API key?

Sign up at [generaltranslation.com](https://www.generaltranslation.com), create a project, and generate API keys from the [API Keys](/docs/platform/projects/api-keys) page in your project settings.

### Can I edit translations after they're generated?

Yes. You can view and edit all translations from the [Translations](/docs/platform/translations) page in the dashboard.

### What are AI context features?

AI context features — [style controls](/docs/platform/ai-context/style-controls), [glossaries](/docs/platform/ai-context/glossary), and [locale context](/docs/platform/ai-context/locale-context) — let you guide how translations are generated to match your brand's tone and terminology.
