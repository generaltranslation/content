Browse Reference pages for the public General Translation API endpoints. See the [Quickstart](/docs/platform/openapi/quickstart) for base URLs, authentication, versioning, and errors.

## Files [#files]

| Page | Description |
| --- | --- |
| [Upload source files](/docs/platform/openapi/reference/files/upload-source) | Upload one or more source files to a Project before queuing translation. |
| [Upload translations](/docs/platform/openapi/reference/files/upload-translations) | Upload translated files that match previously uploaded source files. |
| [Submit file diffs](/docs/platform/openapi/reference/files/submit-diffs) | Submit diffs for changed source or translated content instead of uploading full files. |
| [Download files](/docs/platform/openapi/reference/files/download-many) | Download multiple translated files in one request. |
| [Download file](/docs/platform/openapi/reference/files/download) | Download a single translated file for a Project, locale, or version. |
| [Publish files](/docs/platform/openapi/reference/files/publish-files) | Mark translated files as published so they are ready for delivery. |
| [Get file information](/docs/platform/openapi/reference/files/file-info) | Inspect source file metadata, versions, locales, and translation state. |
| [Get translation status](/docs/platform/openapi/reference/files/translation-status) | Check translation progress and availability by file and locale. |
| [Move files](/docs/platform/openapi/reference/files/move-files) | Move or rename files in a Project. |
| [Get orphaned files](/docs/platform/openapi/reference/files/orphaned-files) | Find translated files that are no longer connected to current source files. |
