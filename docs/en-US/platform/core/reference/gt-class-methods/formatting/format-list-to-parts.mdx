Formats an array into locale-specific parts on a [GT](/docs/platform/core/reference/gt-class/constructor) instance, inserting string separators between items while preserving the original type of each element. General Translation uses the built-in `Intl.ListFormat` API. Unlike [`formatList`](/docs/platform/core/reference/gt-class-methods/formatting/format-list), which returns a flat list string, this method returns an `Array<T | string>`, making it ideal for mixed-type arrays containing strings, numbers, objects, or JSX elements.

## Overview [#overview]

Call `formatListToParts` on a `GT` instance with an array and an optional options object. It returns an array of the original items interleaved with locale-aware string separators.

```typescript
const gt = new GT({ locales: ['en'] });

const parts = gt.formatListToParts(['apple', 42, { type: 'fruit' }]);
// ['apple', ', ', 42, ', and ', { type: 'fruit' }]
```

Signature:

```typescript
formatListToParts<T>(
  array: Array<T>,
  options?: { locales?: string | string[] } & Intl.ListFormatOptions
): Array<T | string>
```

*Note: `formatListToParts` runs locally using `Intl.ListFormat` and does not require an API key. It uses the instance's locales by default; pass `locales` to override. For formatting without a `GT` instance, see the standalone [`formatListToParts`](/docs/platform/core/reference/utility-functions/formatting/format-list-to-parts).*

## How it works [#how-it-works]

- **Type preservation.** Only string separators are inserted between elements; the original item types are kept, which is the key differentiator from a flat list string.
- **Locale resolution.** By default the method formats for the instance's locales. Pass `locales` in the options to override them.
- **Intl-backed.** Formatting is delegated to the browser-native [`Intl.ListFormat`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/ListFormat).
- **List type.** `type: 'conjunction'` produces "and" lists, `'disjunction'` produces "or" lists, and `'unit'` produces unit-style lists.

## Parameters [#parameters]

| Parameter | Description | Type | Optional | Default |
| --- | --- | --- | --- | --- |
| [`array`](#array) | The array of items to format. | `Array<T>` | No | — |
| [`options`](#options) | Formatting configuration, extending `Intl.ListFormatOptions` with a `locales` override. | `object` | Yes | — |

### `array` [#array]

**Type** `Array<T>` · **Required**

The array of items to format. Items may be any type; non-string items are preserved as-is in the output.

### `options` [#options]

**Type** `{ locales?: string | string[] } & Intl.ListFormatOptions` · **Optional**

Formatting configuration. Extends `Intl.ListFormatOptions` with an additional `locales` field:

| Name | Description | Type | Optional | Default |
| --- | --- | --- | --- | --- |
| `locales` | Override locales for formatting. | `string \| string[]` | Yes | instance locales |
| `type` | List formatting type. | `'conjunction' \| 'disjunction' \| 'unit'` | Yes | `'conjunction'` |
| `style` | List formatting style. | `'long' \| 'short' \| 'narrow'` | Yes | `'long'` |

## Returns [#returns]

**Type** `Array<T | string>`

An array of the original items interleaved with locale-specific string separators.

## Examples [#examples]

```typescript
import { GT } from 'generaltranslation';

const gt = new GT({ locales: ['en'] });

// Conjunction (default) — "and" list
console.log(gt.formatListToParts(['A', 'B', 'C']));
// Output: ['A', ', ', 'B', ', and ', 'C']

// Disjunction — "or" list
console.log(gt.formatListToParts(['A', 'B', 'C'], { type: 'disjunction' }));
// Output: ['A', ', ', 'B', ', or ', 'C']
```

```typescript
// Mixed-type arrays — numbers and objects are preserved
console.log(gt.formatListToParts(['apple', 42, { type: 'fruit' }]));
// Output: ['apple', ', ', 42, ', and ', { type: 'fruit' }]
```

```typescript
// Style variations

// Short style
console.log(gt.formatListToParts(['first', 'second'], { style: 'short' }));
// Output: ['first', ' & ', 'second']

// Unit style
console.log(gt.formatListToParts(['1km', '2mi'], { type: 'unit' }));
// Output: ['1km', ', ', '2mi']
```

```typescript
// Locale-specific formatting — Spanish disjunction
console.log(gt.formatListToParts(['red', 'green', 'blue'], {
  locales: ['es'],
  type: 'disjunction',
}));
// Output: ['red', ', ', 'green', ' o ', 'blue']
```

## Notes [#notes]

- Original item types are preserved — only string separators are inserted between elements.
- This is the key differentiator from a flat list string.
- Particularly useful for rendering mixed-type arrays in UI frameworks like React.
- The method uses browser-native `Intl.ListFormat` under the hood.
