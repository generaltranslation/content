title: EntryMetadata

`EntryMetadata` provides optional configuration for a [`TranslateManyEntry`](/docs/platform/core/reference/types/translate-many-entry) object, adding context and controls that customize how General Translation translates the entry.

## Overview [#overview]

`EntryMetadata` is the `metadata` field on an object-form translation entry. Every field is optional.

```typescript
type EntryMetadata = {
  id?: string;
  hash?: string;
  context?: string;
  maxChars?: number;
  dataFormat?: DataFormat;
  actionType?: ActionType;
};
```

The related `ActionType` is a string union:

```typescript
type ActionType = 'fast';
```

## Properties [#properties]

| Property | Description | Type | Optional |
| --- | --- | --- | --- |
| [`id`](#id) | Unique identifier for the entry. | `string` | Yes |
| [`hash`](#hash) | Content hash for caching and deduplication. | `string` | Yes |
| [`context`](#context) | Context hint for translators. | `string` | Yes |
| [`maxChars`](#max-chars) | Maximum character limit for the translation. | `number` | Yes |
| [`dataFormat`](#data-format) | Content format specification. | [`DataFormat`](/docs/platform/core/reference/types/data-format) | Yes |
| [`actionType`](#action-type) | Translation model preference. | `ActionType` | Yes |

### `id` [#id]

**Type** `string` · **Optional**

Unique identifier for the entry.

### `hash` [#hash]

**Type** `string` · **Optional**

Content hash used for caching and deduplication.

### `context` [#context]

**Type** `string` · **Optional**

Context hint for translators, such as `"Button text"` or `"Navigation menu"`.

### `maxChars` [#max-chars]

**Type** `number` · **Optional**

Maximum character limit for the translation.

### `dataFormat` [#data-format]

**Type** [`DataFormat`](/docs/platform/core/reference/types/data-format) · **Optional**

Content format specification: `'JSX'`, `'ICU'`, `'I18NEXT'`, or `'STRING'`. See [`DataFormat`](/docs/platform/core/reference/types/data-format).

### `actionType` [#action-type]

**Type** `ActionType` · **Optional**

Translation model preference. Currently the only accepted value is `'fast'`, which selects the mini model.

## Example [#example]

```typescript
import { GT, TranslateManyEntry } from 'generaltranslation';

const entry: TranslateManyEntry = {
  source: 'Save',
  metadata: {
    context: 'Button text',
    actionType: 'fast'
  }
};

const gt = new GT({ apiKey: 'your-api-key', projectId: 'your-project-id' });
const result = await gt.translate(entry, 'es');
```

```typescript
// With ICU format
const entry: TranslateManyEntry = {
  source: '{count, plural, other {{count} items}}',
  metadata: {
    dataFormat: 'ICU',
    context: 'Item count'
  }
};
```
