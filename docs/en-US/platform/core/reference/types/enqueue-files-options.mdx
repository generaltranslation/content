`EnqueueFilesOptions` is the configuration type passed to [`enqueueFiles`](/docs/platform/core/reference/gt-class-methods/translation/enqueue-files) to queue previously uploaded source files for translation.

*Note: this is the user-facing type accepted by [`enqueueFiles`](/docs/platform/core/reference/gt-class-methods/translation/enqueue-files) in 9.0.*

## Overview [#overview]

`EnqueueFilesOptions` describes the target locales and job settings for a batch file translation. Only `targetLocales` is required; every other field is optional.

```typescript
type EnqueueFilesOptions = {
  sourceLocale?: string;
  targetLocales: string[];
  modelProvider?: string;
  force?: boolean;
  timeout?: number;
};
```

## Properties [#properties]

| Property | Description | Type | Optional |
| --- | --- | --- | --- |
| [`sourceLocale`](#source-locale) | Source locale override. | `string` | Yes |
| [`targetLocales`](#target-locales) | Target locale codes to translate into. | `string[]` | No |
| [`modelProvider`](#model-provider) | Translation model provider. | `string` | Yes |
| [`force`](#force) | Invalidate cached translations and force re-translation. | `boolean` | Yes |
| [`timeout`](#timeout) | Request timeout in milliseconds. | `number` | Yes |

*Note: `requireApproval` was an 8.x-only option and is not present in 9.0.*

### `sourceLocale` [#source-locale]

**Type** `string` · **Optional**

Overrides the source locale for this job. Defaults to the instance `sourceLocale` when omitted.

### `targetLocales` [#target-locales]

**Type** `string[]` · **Required**

The locale codes to translate the files into.

### `modelProvider` [#model-provider]

**Type** `string` · **Optional**

The translation model provider to use.

### `force` [#force]

**Type** `boolean` · **Optional**

Invalidate cached translations and force re-translation of files even if translations already exist.

### `timeout` [#timeout]

**Type** `number` · **Optional**

Request timeout in milliseconds.

## Example [#example]

```typescript
import { GT, EnqueueFilesOptions } from 'generaltranslation';

const options: EnqueueFilesOptions = {
  targetLocales: ['es', 'fr', 'de'],
  sourceLocale: 'en',
  modelProvider: 'premium',
};

const result = await gt.enqueueFiles(files, options);
```

```typescript
// Development vs production
const devOptions: EnqueueFilesOptions = {
  targetLocales: ['es'],
};

const prodOptions: EnqueueFilesOptions = {
  targetLocales: ['es', 'fr', 'de', 'ja'],
  modelProvider: 'premium',
  force: true,
};
```
