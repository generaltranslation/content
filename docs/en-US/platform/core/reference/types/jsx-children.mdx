title: JsxChildren
description: JSX content structure for translatable children. API reference for JsxChildren type.

`JsxChildren` represents JSX content — text, structured elements, and variables — that General Translation can translate and render. It is the structured form of [`Content`](/docs/platform/core/reference/types/content) used for the `'JSX'` [`DataFormat`](/docs/platform/core/reference/types/data-format).

## Overview [#overview]

`JsxChildren` is either a single child or an array of children. Each child is a `JsxChild`: a string, a [`JsxElement`](/docs/platform/core/reference/types/jsx-element), or a [`Variable`](/docs/platform/core/reference/types/variable).

```typescript
type JsxChildren = JsxChild | JsxChild[];

type JsxChild = string | JsxElement | Variable;
```

## Child types [#child-types]

A `JsxChild` is one of the following:

| Type | Description |
| --- | --- |
| `string` | Plain text content. |
| [`JsxElement`](/docs/platform/core/reference/types/jsx-element) | Structured element with a tag, attributes, and children. |
| [`Variable`](/docs/platform/core/reference/types/variable) | Dynamic placeholder for runtime content. |

### `string`

Plain text content, rendered as-is.

### `JsxElement`

A structured HTML-like element with an optional tag (`t`), id (`i`), GT properties (`d`), and children (`c`). See [`JsxElement`](/docs/platform/core/reference/types/jsx-element) for the full shape:

```typescript
type JsxElement = {
  t?: string;      // tag name
  i?: number;      // id
  d?: GTProp;      // GT properties
  c?: JsxChildren; // children
};
```

### `Variable`

A placeholder for dynamic content, identified by a key (`k`) with an optional formatting type (`v`). See [`Variable`](/docs/platform/core/reference/types/variable).

## Example [#example]

```typescript
import { JsxChildren, Variable } from 'generaltranslation/types';

// Simple text
const text: JsxChildren = "Welcome!";

// Text with variables
const greeting: JsxChildren = [
  "Hello, ",
  { k: 'userName' } as Variable,
  "!"
];
```

```typescript
// Div element
const divElement: JsxChildren = {
  t: 'div',
  c: ['Content here']
};

// Link with title
const linkElement: JsxChildren = {
  t: 'a',
  d: { ti: 'Visit homepage' },
  c: ['Click here']
};
```
