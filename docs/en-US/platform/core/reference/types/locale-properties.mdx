title: LocaleProperties

`LocaleProperties` provides detailed linguistic and regional information about a locale. It is returned by the General Translation [`getLocaleProperties`](/docs/platform/core/reference/gt-class-methods/locales/get-locale-properties) method.

## Overview [#overview]

`LocaleProperties` describes a locale across its language, script, and region dimensions, giving both source-language and native-language names for each, plus maximized and minimized forms and a flag emoji.

```typescript
type LocaleProperties = {
  code: string;
  name: string;
  nativeName: string;
  languageCode: string;
  languageName: string;
  nativeLanguageName: string;
  nameWithRegionCode: string;
  nativeNameWithRegionCode: string;
  regionCode: string;
  regionName: string;
  nativeRegionName: string;
  scriptCode: string;
  scriptName: string;
  nativeScriptName: string;
  maximizedCode: string;
  maximizedName: string;
  nativeMaximizedName: string;
  minimizedCode: string;
  minimizedName: string;
  nativeMinimizedName: string;
  emoji: string;
};
```

*Note:* the older docs typed this as an `interface`; the current source defines it as a `type` alias.

## Properties [#properties]

Every property is a `string` and always present on the returned object.

| Property | Description | Type |
| --- | --- | --- |
| [`code`](#code) | Original locale code. | `string` |
| [`name`](#name) | Display name in the source language. | `string` |
| [`nativeName`](#native-name) | Display name in the native language. | `string` |
| [`languageCode`](#language-code) | Base language code. | `string` |
| `languageName` | Language name in the source language. | `string` |
| `nativeLanguageName` | Language name in the native language. | `string` |
| `nameWithRegionCode` | Display name including the region code. | `string` |
| `nativeNameWithRegionCode` | Native display name including the region code. | `string` |
| [`regionCode`](#region-code) | ISO region code. | `string` |
| `regionName` | Region name in the source language. | `string` |
| `nativeRegionName` | Region name in the native language. | `string` |
| [`scriptCode`](#script-code) | ISO script code. | `string` |
| `scriptName` | Script name in the source language. | `string` |
| `nativeScriptName` | Script name in the native language. | `string` |
| [`maximizedCode`](#maximized-code) | Locale with likely script and region added. | `string` |
| `maximizedName` | Display name of the maximized locale. | `string` |
| `nativeMaximizedName` | Native display name of the maximized locale. | `string` |
| [`minimizedCode`](#minimized-code) | Shortest valid locale code. | `string` |
| `minimizedName` | Display name of the minimized locale. | `string` |
| `nativeMinimizedName` | Native display name of the minimized locale. | `string` |
| [`emoji`](#emoji) | Flag emoji. | `string` |

### `code` [#code]

**Type** `string` · **Required**

The original locale code.

### `name` [#name]

**Type** `string` · **Required**

Display name in the source language, for example `"Austrian German"`.

### `nativeName` [#native-name]

**Type** `string` · **Required**

Display name in the native language, for example `"Österreichisches Deutsch"`.

### `languageCode` [#language-code]

**Type** `string` · **Required**

The base language code.

### `regionCode` [#region-code]

**Type** `string` · **Required**

ISO region code.

### `scriptCode` [#script-code]

**Type** `string` · **Required**

ISO script code.

### `maximizedCode` [#maximized-code]

**Type** `string` · **Required**

The locale with its likely script and region added.

### `minimizedCode` [#minimized-code]

**Type** `string` · **Required**

The shortest valid locale code.

### `emoji` [#emoji]

**Type** `string` · **Required**

Flag emoji for the locale.

## Example [#example]

```typescript
const gt = new GT({ sourceLocale: 'en-US' });
const props = gt.getLocaleProperties('de-AT');

console.log(props.name);        // "Austrian German"
console.log(props.nativeName);  // "Österreichisches Deutsch"
console.log(props.emoji);       // "🇦🇹"
```
