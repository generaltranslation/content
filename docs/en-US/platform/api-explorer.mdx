Authenticate every request with an API key in the `x-gt-api-key` header. Organization keys must also send `x-gt-project-id` on project-scoped routes. See [API keys](/docs/platform/api-keys) for how to create and scope keys, and the [API reference](/docs/platform/api-reference) for base URLs, versioning, and rate limits.
