## Locale routing

`gt-tanstack-start` now supports opt-in locale routing. Enable `localeRouting` in `gt.config.json` to prioritize locale path prefixes, keep the default locale unprefixed, and update the URL automatically when the locale changes.

Custom `_reload` behavior still takes precedence when configured.
