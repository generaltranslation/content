## Overview

The `useSetLocale` hook is used to set the user's locale.
This is client-side only, and it must be wrapped in a [`<GTProvider>`](/docs/react/reference/components/gt-provider) component.

## Reference

### Returns

A function that sets the user's locale.

---

## Examples

### Basic usage

```jsx
import { useSetLocale } from '__PACKAGE_NAME__';

export default function MyComponent() {
  const setLocale = useSetLocale();

  return <button onClick={() => setLocale('en')}>Set Locale</button>;
}
```
